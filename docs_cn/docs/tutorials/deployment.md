# 部署

开始使用 [LangGraph Platform](../concepts/langgraph_platform.md) 在本地或云端部署你的 LangGraph 应用程序。

## 快速开始 🚀 {#quick-start}

- [LangGraph Server 快速入门](../tutorials/langgraph-platform/local-server.md)：在本地启动 LangGraph 服务器，并使用 REST API 和 LangGraph Studio Web UI 与其交互。
- [LangGraph 模板快速入门](../concepts/template_applications.md)：使用模板应用程序开始构建 LangGraph Platform。
- [使用 LangGraph Cloud 快速入门部署](../cloud/quick_start.md)：使用 LangGraph Cloud 部署 LangGraph 应用程序。


## 部署选项

- [Self-Hosted Lite](../concepts/self_hosted.md)：LangGraph Platform 的免费（最多 100 万个节点执行）、有限版本，你可以在本地或以自托管方式运行
- [Cloud SaaS](../concepts/langgraph_cloud.md)：作为 LangSmith 的一部分托管。
- [Bring Your Own Cloud](../concepts/bring_your_own_cloud.md)：我们管理基础设施，所以你无需操心，但所有基础设施都在你的云中运行。
- [Self-Hosted Enterprise](../concepts/self_hosted.md)：完全由你管理。
