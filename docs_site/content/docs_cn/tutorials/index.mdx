# 教程

欢迎来到 LangGraph.js 教程！这些笔记通过构建各种语言代理和应用程序来介绍 LangGraph。

## 快速入门

通过全面的快速入门学习 LangGraph 的基础知识，在其中你将从零开始构建一个代理。

- [快速入门](quickstart.ipynb)
- [常见工作流](workflows/index.md)：使用 LLM 通过 LangGraph 实现的最常见工作流概述。
- [LangGraph Cloud 快速入门](/langgraphjs/cloud/quick_start/)：在本教程中，你将构建一个代理并将其部署到 LangGraph Cloud。
